Scayt spellchecker предлагает три варианта Конфигурации: on, off, disabled. На данный момент настройкой по умолчанию будет **off** вместо **on**. С помощью **on** содержимое редактора анализируется Scayt и отправляется для проверки орфографии на сервер Scayt. Хотя нет никаких оснований полагать, что это представляет угрозу безопасности, тем не менее это изменение делается из соображений пользователей с частными вики-сайтами, которые могут испытывать беспокойство из-за отправки своих данных на удаленный сервер. Дополнительную информацию можно найти на странице Параметры конфигурации документации ckgedit по адресу https://www.dokuwiki.org/plugin:ckgedit:configuration#scayt_lang.
